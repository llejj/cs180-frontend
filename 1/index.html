Project 1: Colorizing the Prokudin-Gorskii Photo Collection

1. Background

Sergei Mikhailovich Prokudin-Gorskii (1863-1944) was a pioneering photographer who developed a unique method for creating color photographs in the early 1900s. His technique involved capturing three separate exposures of each scene through red, green, and blue filters. These images were recorded onto glass plates as a single, vertically-stacked black and white negative.

This project aims to digitally process and combine these historical glass plate images to reconstruct their original color appearance. The challenge lies in properly aligning the three color channels, as even slight misalignments can result in color artifacts and blurring in the final image. We will assume that a simple x,y translation model is sufficient for proper alignment.

2. Single-Scale Image Alignment

2.1 Implementation

The single-scale approach implements a straightforward alignment method with the following steps:

  • Splits the input image into three equal parts (B, G, R channels)
  • Uses the blue channel as the reference
  • Aligns the green and red channels to the blue channel using either:
    • L2 norm (Euclidean distance)
    • Normalized Cross-Correlation (NCC)
  • Searches for the best alignment within a [-15, 15] pixel window

Alignment Metrics

Two alignment metrics were implemented and compared:

  1. L2 Norm (Euclidean Distance):

    Computed as sqrt(sum((image1-image2)^2)). This metric is simple but can be sensitive to intensity differences between channels.

  2. Normalized Cross-Correlation (NCC):

    Implemented as a normalized dot product between the images. This metric proved more robust, especially for images with different channel intensities.

3. Multi-Scale Pyramid Alignment

3.1 Implementation

To handle the larger TIF images efficiently, a multi-scale pyramid approach was implemented with the following key features:

  • Recursively downscales images by a factor of 2
  • Starts alignment at the coarsest level
  • Refines alignment at each subsequent level
  • Searches for the best alignment within a [-8, 8] pixel window around the scaled-up alignment from the previous level

Optimization Techniques

  • Edge cropping (10%) during alignment to reduce edge effects
  • Pyramid-based coarse-to-fine alignment for efficient processing of large images
  • Adaptive search windows based on previous pyramid level results

4. Conclusions

The implemented algorithm successfully colorized the Prokudin-Gorskii photo collection using two different approaches:

  • Single-Scale Alignment: Provided excellent results for smaller images (monastery.jpg, cathedral.jpg) using direct pixel-wise alignment with NCC metric.
  • Multi-Scale Pyramid: Effectively handled larger TIF files through efficient coarse-to-fine alignment, though some challenging cases like emir.tif required special handling due to intensity variations.

The combination of NCC metric with pyramid-based alignment proved robust and computationally efficient, successfully processing images of various sizes and characteristics. The adaptive search windows and edge cropping optimizations further improved the quality of the results.
